# Widget Ideas for Lab Dash

A collection of potential widgets to build for Lab Dash.

---

## Monitoring & System Health

### 1. Docker Container Status
Show running/stopped containers, resource usage, quick start/stop controls.

### 2. Uptime Monitor
Track uptime/downtime history for multiple services with visual timeline.

### 3. Log Viewer
Display recent log entries from selected services/containers.

### 4. Speedtest Widget
Run periodic internet speed tests and display download/upload/ping history.

---

## Network & Security

### 5. DNS Query Stats
Show DNS resolution times, cache hit rates (for non-Pi-hole DNS servers).

### 6. VPN Status
Display connected VPN status, IP location, connection time.

### 7. Firewall/Fail2ban Stats
Show blocked IPs, ban counts, recent threats.

### 8. SSL Certificate Monitor
Track certificate expiration dates for your domains.

---

## Media & Entertainment

### 9. Plex/Emby Widget
Currently playing, library stats, active streams (alternative to Jellyfin).

### 10. Audiobook Progress (Audiobookshelf)
Currently listening, progress bars.

### 11. Podcast Stats
Recent episodes, subscriptions, listening history.

---

## Home Automation

### 12. Home Assistant Widget
Quick toggles for lights/switches, sensor readings.

### 13. Zigbee/Z-Wave Device Status
Show connected devices and battery levels.

### 14. Thermostat Widget
Current temp, setpoint, quick adjustments.

---

## Productivity & Info

### 15. Calendar Widget
Upcoming events from CalDAV/Google Calendar.

### 16. RSS Feed Widget
Latest headlines from configured feeds.

### 17. Git Repository Stats
Recent commits, open PRs/issues for your repos.

### 18. Cryptocurrency Prices
Track selected coins with price charts.

---

## Storage & Backup

### 19. NAS Storage Widget
Pool health, space usage, active transfers.

### 20. Backup Status
Last backup time, success/failure status (Restic, Borg, etc.).
